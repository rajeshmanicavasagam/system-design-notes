# System-Design-Notizen (Enterprise & Cloud)

Dieses Repository enthält praxisnahe Notizen zur Systemarchitektur
aus realen Enterprise-Projekten mit .NET und Cloud-Plattformen.

Der Fokus liegt auf:
- Architekturentscheidungen und Trade-offs
- Wartbarkeit und Skalierbarkeit
- pragmatischen Lösungen statt Theorie

Diese Notizen dienen der strukturierten Entscheidungsfindung
in langlebigen Softwaresystemen.
